# **App Name**: StyleAI

## Core Features:

- Writing Sample Upload: Users can upload their writing samples (PDFs) to establish their writing style.
- AI Style Analysis: Analyze user-uploaded writing samples using Google's Gemini API to create a custom writing profile; acts as a tool when generating tailored content.
- AI Content Generation: Generate new content tailored to the user's unique style, powered by their analyzed writing profile, using Google Gemini.
- Style Refinement: Users can fine-tune their style profiles based on generated content to continuously refine AI outputs.
- Content Management: Dashboard for creating, managing, and organizing user content.

## Style Guidelines:

- Primary color: HSL(48, 100%, 50%) translating to a vibrant yellow (#FFDA63) to evoke creativity and intelligence.
- Background color: Desaturated yellow, approximately HSL(48, 20%, 95%) translating to a very light yellow (#F8F8F1) to provide a subtle, non-distracting backdrop.
- Accent color: Analogous green-yellow, approximately HSL(18, 100%, 50%) translating to bright orange (#FF8833), used sparingly for call-to-action buttons and important notifications, drawing user attention without overwhelming the palette.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look; suitable for headlines or body text.
- Crisp, minimalist icons to represent various content types and functionalities. Use consistent line weights and a restricted color palette (using only primary and accent colors) to maintain visual harmony.
- Use a modular grid layout with clear content separation. Prioritize readability and user flow to prevent any cognitive overload.
- Subtle transitions and animations on UI elements to signal changes and provide feedback, enhancing the user experience without being obtrusive.
